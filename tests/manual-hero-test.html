Hero Movement Manual Test Instructions

Test 1: Hero Movement Before Game Start

  1. Load the game at http://localhost:3000
  2. Select a hero type and click Level 1
  3. Verify the "Start" button is visible, indicating game hasn't started
  4. Click on the hero to select it (it should highlight)
  5. Click somewhere else on the map
  6. The hero should start moving to the clicked location
Expected: Hero moves to clicked location, even though game hasn't started.

Test 2: Hero Deselection After Destination Set

  1. After Test 1, click on the hero again to select it
  2. Click somewhere else on the map
  3. The hero should no longer be highlighted (deselected)
Expected: Hero is automatically deselected after setting movement destination.

Test 3: Hero Continues Moving After Deselection

  1. Click on the hero to select it
  2. Click on a far location on the map
  3. Observe the hero for several seconds
Expected: Hero continues moving toward destination even after being deselected.

If Any Test Fails:

Check the browser console (F12) for error messages.

Look for logs about movement and gather points:

  • Canvas click at: - Shows where you clicked
  • Setting gather point for hero - Should show when you click to move
  • Hero moved X units - Shows movement progress
